Projeto Final parte 01

Codigo que representa um simples carrinho de E-commerce

Orientações

Aperte "ctrl + shift + i" para abrir o console do browser

  • Use o camando "let nome_produto = new Produto(identificação, nome, preço)" par criar o novo produto, passando o nome do objeto, e os parametros desejados para Produto()

  • Digite o camando "nome_produto.cartItem(quantidade)"" para adicionar a quantidade de produtos desejada. Será impresso o valor total dos produtos.

  • Para adicionar o produto ao carrinho digite o comando "cart.addItem(nome_produto)"

  • Para remover um dos produto do carrinho digite "cart.removeItem(numero_indice), passando como parametro o indicice do carrinho."
